####################		QueryString			####################

1、QueryString["Composite_FieldName"]			联合主键字段名	
示例：Composite_FieldName = 'EnterpriseCode'

2、QueryString["Composite_FieldValue"]			联合主键字段值	
示例：Composite_FieldValue = 'E0000001'

3、QueryString["config_Id"]						单表采集主配置项	
示例：config_Id = 1005

4、QueryString["fileNameColumn"]				文件名列
示例：fileNameColumn = Document_Name

5、QueryString["imgColumn"]						二进制数据列		
示例：imgColumn=Document_Data

6、QueryString["imgFormat"]						二进制数据格式列	
示例：imgFormat=imgFormat


####################		Session				####################

1、Session["Query"]								查询条件
2、Session["userID"]							用舻锹济
3、Session["userName"]							用户真实姓名
4、Session["userRight"]							用户权限
